# Graduation Project

A native Android application built as a graduation project, implementing a chatbot experience. Written in Java and Kotlin using the standard Android Studio/Gradle project structure.

## Project structure

- android/ - the Android application module (Gradle build files, app source under android/app/src)
- assets/ - project assets used by the app

## Tech stack

- Java and Kotlin
- Gradle build system (Android Gradle Plugin)

## Getting started

1. Open the android folder in Android Studio.
2. Let Gradle sync and download dependencies.
3. Run the app on an emulator or physical device.
